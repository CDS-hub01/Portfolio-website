# Christian Sokari — Frontend Developer Portfolio

A clean, modern, and fully responsive personal portfolio website built with pure HTML, CSS and JavaScript. Designed to showcase real-world projects, technical skills, and professional experience.

## 🌐 Live Site

[https://cds-portfolio.netlify.app](https://cds-portfolio.netlify.app)

## 📌 About This Portfolio

This portfolio was built from scratch without any frameworks or libraries — just clean, purposeful HTML, CSS and JavaScript. Every design decision was intentional, from the Dark Slate + Indigo color palette to the varied section heading styles across the page.

## ✨ Features

- Fully responsive across mobile, tablet and desktop
- Smooth hamburger navigation (slides in from right on mobile)
- Active nav link highlighting on scroll
- Sticky navbar with blur backdrop effect
- Animated hero section with decorative code block
- Project cards with hover overlay and live/code links
- Skills grid with "Currently Learning" badges
- Direct contact links (Email, LinkedIn, GitHub, WhatsApp, Phone)
- CV download button linked to Google Drive PDF
- Auto-updating footer year

## 🛠️ Built With

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Syne + Inter)
- Deployed on Netlify
